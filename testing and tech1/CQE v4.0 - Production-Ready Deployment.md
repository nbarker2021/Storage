### Core Architecture

The CQE system operates on three fundamental principles:

1. **Geometry First, Meaning Second** - All data is processed based on geometric structure, with semantic meaning emerging from stable geometric configurations
2. **E8 and Toroidal Closure** - Universal atomization into 8-dimensional E8 lattice space with toroidal projection for rotational analysis
3. **Dihedral Symmetry Governance** - All transformations constrained by dihedral groups (D_n) and Cartan-form enforced order

## Quick Start

```python
from cqe.core.cqe_complete_system import CQEKernal
from cqe.modules.ultimate_unified_cqe_system import UniversalAtomFactory

# Initialize the CQE kernel
kernel = CQEKernal()

# Create a universal atom
factory = UniversalAtomFactory()
atom = factory.create_from_data("Sample data")

# Process through geometric reasoning
result = kernel.process(atom)

print(f"Closure State: {result.closure_state}")
print(f"Weyl Chamber: {result.weyl_chamber}")
print(f"Geometric Properties: {result.properties}")
```
