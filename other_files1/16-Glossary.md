# Glossary

- **AGRM** — Adaptive Golden Ratio Method stack (MDHG hash + controller).
- **MDHG** — Multi‑Dimensional Hierarchical Grid; hash table layout.
- **SNAP** — Typed state store for personas, code, ops artifacts.
- **SNAPDNA** — Persona record with expertise level and meta.
- **SNAPBIT** — Minimal, testable code unit (CodeUnit).
- **DTT** — Deploy‑to‑Test; gated adoption policy.
- **Bouncing Batch** — Superpermutation parallel evaluation per cell.
- **Laminate** — Graph of allowed transitions; anti‑laminate disallows.
