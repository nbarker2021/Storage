
# Morphonic–CQE Unified Build v1

This package unifies provided core, sidecars, personal node app, experiments, and docs.

## Layout
- `morphonic_cqe_unified/core/` — math & governance primitives
- `morphonic_cqe_unified/sidecar/` — original and **SpeedLight V2 (Plus)** + adapter
- `morphonic_cqe_unified/apps/` — **cqe_personal_node** (patched for packaged imports)
- `morphonic_cqe_unified/experimental/` — lambda calculus + geometric transformers
- `docs/` — whitepapers and reports supplied
- `examples/` — quickstart launcher

## Install (editable)
```bash
pip install -e .
cqe-personal-node
```
