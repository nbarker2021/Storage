## Concept and Design Section

### In Silico vs In Vitro Learning Paradigms

The article makes a crucial distinction between two learning paradigms:

#### In Silico Learning:
- Produces DNA neural network with **fixed memories** 
- Acts like a **hardwired processor**
- Memories are predetermined and static

#### In Vitro Learning (This Work):
- Enables the **liquid system itself** to form memories
- Responds to example molecular inputs during training phase
- Acts like an **adaptive memory device**
- **Integrated system** provides learned input-output functions for subsequent testing

### Technical Implementation Details

#### Memory Formation Process:
1. **Initial State**: All weights are zero (blank memory)
2. **Training Phase**: Input patterns and class labels collectively activate specific weights
3. **Memory Encoding**: Input added to corresponding memory through molecular concentrations
4. **Weight Storage**: Resulting weights encoded in concentrations of activator molecules
5. **Transition Phase**: Learning transitions to testing by connecting memory device to processor
6. **Information Transfer**: Information flows from activators to weight molecules
7. **Active Weights**: Now-active weights interact with test inputs, compute weighted sums
8. **Classification**: Winner-take-all competition produces binary signals that classify input

### Key Innovation: Activatable Memory System

The system demonstrates a novel **activatable memory** concept where:
- Memories are not pre-programmed but **formed during training**
- **Molecular concentrations** serve as the memory storage medium
- **Chemical reactions** perform both learning and inference
- **Structural changes** in the molecular network encode learned information

### Implementation of Learning and Testing

The article details the implementation of the learning and testing phases using DNA molecules:

#### Testing Phase (Builds on previous work):
- **Weighted Sum Function**: Divided into weight multiplication and summation
- **Catalytic and Stoichiometric Reactions**: Weight multiplication is catalytic, summation is stoichiometric
- **Fan-out/Fan-in**: Implemented using amplification gate motif (fan-out) and integration gate motif (fan-in)
- **Thresholding Reaction**: Cleans up noisy input signals, converts corrupted inputs to ideal binary signals
- **Winner-Take-All Function**: Implemented using pairwise annihilation and signal restoration
- **Reporting Reaction**: Converts output signal to fluorescence for readout

#### Training Phase (Novel Development):
- **Key Function**: Summed multiplication of binary input and label signals
- **Challenge**: Product species (weights) are double-stranded gate species, not signal species
- **Solution**: Developed an **activatable amplification gate motif** to translate signal species into gate species
- **Two Reactions for Training**: 
    1. **Supervised Learning**: Input and label strand collectively produce an activator (Act_i,j_)
    2. **Weight Activation**: Activator turns on a specific weight gate (W_i,j_)
- **Activatable Transformation Gate Motif**: Used for the supervised learning reaction
- **Scaling**: Implicitly achieved by using lower amount of input strand per training pattern for more patterns
- **Label Inhibition Reaction**: Cleans up excess label between training events for accurate learning

#### Five-Layer DNA Neural Network:
- Combines training and testing phases
- First two layers utilize two types of activatable gate motifs
- Output of one is an activator signal, the other is a regular signal

### Motif Characterization

The article provides detailed characterization of the key molecular motifs:

#### Activatable Amplification Gate:
- **Function**: Catalytically produces a weighted input signal (P_i,j_)
- **Structure**: Consists of a toehold 'T' flanked by two long domains Xi and Pj
- **Activation**: Activator (Act_i,j_) must carry both input bit (i) and memory class (j) information
- **Design Improvement**: Encodes class information in toehold Tj and bit information in branch migration domain Ai
- **Mechanism**: Activator with matching Tj and Ai binds to the inhibited weight (W*_i,j_), exposing a universal toehold U* for input binding
- **Thermodynamic Drive**: Bulge loop B between U and Xi in the weight gate's top strand, with the activator carrying B*
- **Bulge Elimination**: Lowers the reverse rate and prolongs U* exposure, enhancing input binding
