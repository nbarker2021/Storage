## Quick Start

```python
from cqe import UniversalAtomFactory, UniversalStorage, CombinationEngine

# Initialize components
factory = UniversalAtomFactory()
storage = UniversalStorage()
combiner = CombinationEngine()

# Create Universal Atoms from any data
atom1 = factory.create_atom(432)  # Number
atom2 = factory.create_atom("sacred geometry")  # Text
atom3 = factory.create_atom([1, 1, 2, 3, 5, 8])  # Sequence

# Store atoms
storage.store(atom1)
storage.store(atom2)
storage.store(atom3)

# Search by properties
freq_atoms = storage.find_by_frequency(528.0)  # Find by sacred frequency
root_atoms = storage.find_by_digital_root(9)   # Find by digital root

# Combine atoms
valid_types = combiner.can_combine(atom1, atom2)
if valid_types:
    combined = combiner.combine(atom1, atom2, valid_types[0])
    storage.store(combined)
```
