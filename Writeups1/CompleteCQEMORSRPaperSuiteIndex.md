## Unified Notation Guide

### **Core Mathematical Objects**
- \(\mathfrak{h} \subset \mathfrak{e}_8\): 8-dimensional Cartan subalgebra  
- \(\mathbf{v} = (v_1, \ldots, v_8) \in \mathfrak{h}\): Cartan coordinates
- \(\mathbf{R} = \{\alpha_1, \ldots, \alpha_{240}\}\): E₈ root system
- \(\Delta = \{\alpha_1, \ldots, \alpha_8\}\): Simple root system
- \(\mathcal{W}\): Weyl group of E₈
- \(D_8\): Dihedral group acting on coordinates

### **CQE Framework Components**
- \(\Phi(\mathbf{v}) = \sum_{i=1}^7 w_i(t) \phi_i(\mathbf{v})\): Composite objective function
- \(\{c_i(\mathbf{v})\}_{i=1}^8\): Policy channel coefficients  
- \(\{\mathbf{e}_i\}_{i=1}^8\): Harmonic basis vectors
- \(\mathcal{A} = \{A_1, \ldots, A_k\}\): ALENA operator set
- \(\mathcal{L} = (L_1, \ldots, L_8)\): Lane saturation levels

### **Optimization and Constraints**
- \(\tau_{\text{sat}}, \tau_{\text{conv}}\): Saturation and convergence thresholds
- \(\mathcal{C}_{E_8}\): E₈ lattice-constrained feasible region
- \(\mathbf{g}(\mathbf{v}) \leq \mathbf{0}\): Inequality constraints
- \(\mathbf{h}(\mathbf{v}) = \mathbf{0}\): Equality constraints
